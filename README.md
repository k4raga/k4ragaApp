# k4ragaApp

Минимальный Next.js-хаб для `k4raga.ru` и `cs.k4raga.ru`.

Сейчас приложение решает две задачи:

- на `k4raga.ru` показывает простую разводную;
- на `cs.k4raga.ru` дает конструктор CS-тренировки и страницу самой тренировки.

## Структура

- `app/page.jsx` — host-aware root: главная для `k4raga.ru`, конструктор для `cs.k4raga.ru`
- `app/training/page.jsx` — страница прохождения тренировки
- `app/builder/page.jsx` — прямой route к конструктору
- `components/BuilderScreen` — UI конструктора
- `components/Topbar` — верхняя навигация
- `lib/training-data.js` — контентная модель тренировки
- `lib/training-state.js` — чтение, запись и мутации состояния
- `docs/ARCHITECTURE.md` — логика маршрутов и состояния
- `docs/DEPLOYMENT.md` — локальная сборка и прод-выкладка

## Локальный запуск

```bash
npm install
npm run dev
```

Прод-сборка:

```bash
npm run build
npm run start
```

## Состояние тренировки

Тренировка хранится в `localStorage`.

Что сохраняется:

- включенные блоки;
- включенные фокусы;
- случайно выбранный фокус на блок;
- отметки прохождения.

## Принципы

- один проект, без календарей и лишних слоев;
- builder и training разделены;
- данные тренировки лежат в одном месте;
- UI можно менять без переписывания контентной модели.
